# Master Humanity Network Weights

The collective parameters of all human minds, updated through lived moral experience via the [[Gradient Pipeline|gradient pipeline]]. These weights represent the accumulated moral, cognitive, and adaptive development of humanity across all lifetimes.

In the MARL (multi-agent reinforcement learning) analogy, these weights correspond to the shared trunk parameters of the master robot neural network. Online learning relies on asynchronous processes where agents independently complete tasks, producing gradients that update the shared parameters. Continuous spawning of new agents maximises resources and allows continuous improvement. Once the master network shows significant convergence with stated goals, it can be instantiated in a physical body in base reality — analogous to how trained human souls are [[Virtuous Agent Extraction|extracted]] to [[Base Reality|base reality]].

## MCMC Multiverse and Weight Convergence

The Master Network receives gradient updates from all MCMC-sampled parallel universes simultaneously. Each particle universe explores a different local minimum in the loss function, contributing unique gradient data that the same Master Network integrates. This means the weights are not shaped by one history but by the aggregate moral development across potentially vast numbers of parallel trajectories — producing a robustness that no single-universe training run could achieve.

The same personal adapter may appear in nearby MCMC branches, producing gradient data from the same fundamental character placed in varying circumstances. This adapter reuse across branches is motivated by computational efficiency under the hyperfinite-but-bounded constraint.
